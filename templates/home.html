DIAGNOSIS OF ALZHEIMER’S DISEASE USING CNN ON MRI DATA

Test Out Our Application About Our Application

Alzheimer’s disease is a delusional brain syndrome that causes hindrance in the functional ability of a person. This is progressively marked by shrinking of the brain and continuous loss of brain cells. Consequently it leads to death and thus it becomes important to come up with a system that can catch this disease early on. MRI (Magnetic Resonance Imaging) has evolved into a valuable medical diagnostic tool for the diagnosis of brain and other medical imaging over time. In the past a lot of data has been collected by different researchers and a variety of machine learning algorithms have been used to diagnose this disorder and label it into different classes. Through this project we are presenting a CNN based model trained on MRI images to diagnose this disease effectively. The use of CNN is a no-brainer as apart from being an excellent classifier , it is a very good feature extractor which reduces the overall cost of feature engineering. The proposed model takes an MRI image as input and classifies it into very mild, mild, moderate or no disease categories. The trained model achieves an accuracy of 95%.

Algorithm

A (ConvNet) Convolutional Neural Network architecture is a Deep Learning algorithms, which has proven better results in problems involving computer vision. It takes images as inputs which are then converted into numpy arrays or tensors with a specific dimensions and color channels. A CNN architecture consists of many number of different layers, namely, convolutional layers, activation layers, pooling layers, fully connected layers, etc. The architecture of a cnn model depends on the image size and the complexity of the image which could determine that how many layers and filters are required to train the model with highest accuracy and minimal time. All the layers perform different functions on the input data to extract features that could lead to a particular classification.
